Old Faithful

Using New Project -> Shiny Web App creates a living, breathing Shiny App

A better way to do it

  1. Open a new R Script file.
  2. Start typing shinyapp and press Tab to autocomplete.
  • This will expand into a snippet of codethe skeleton of a Shiny App.
  1. Save the file in Sandbox and run the app.


Its still a working Shiny appits just doesnt do anything. Starting from the snippet is less error-prone than creating a new project and deleting the guts.

Stop an app by clicking the STOP button in the console.
